## 下記for文の`$i = 0`, `$i <= 4`, `$i++`, `echo $i`がそれぞれ、どの順番で処理されるか、また、何をしているのかを説明してください。

```
for ($i = 0; $i <= 4; $i++) {
    echo $i;
}
```

1. $i =0が、最初の1回目の処理で処理される。
2. その次に$i <= 4で$iが条件を満たしているかどうかの処理がされる。
3. 2の処理で条件を満たしていれば、echo $iの処理がされる。
4. 最後に$i++でそのときの$iの値が代入され、次の実行がされる。
